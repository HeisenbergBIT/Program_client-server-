# Program_client-server

Program typu klient-serwer wykorzystujący gniazda sockety z możliwością obsługi wielu klientów w systemie Linux.

Kompilacja programu:
```commandline
gcc server.c -o server
```
```commandline
gcc client.c -o client
```

Uruchomienie programu:
```commandline
./server      ./client
```
